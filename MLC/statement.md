### 题面

**题目名称**：最长公共前缀的总和

**题目描述**  
给定 $ n $ 个由小写英文字母组成的字符串 $ s_1, s_2, \dots, s_n $，所有字符串的总长度不超过 $ 10^6 $。请计算所有无序对 $ (i, j) $（其中 $ i < j $）的最长公共前缀（LCP）的长度之和。

**输入格式**  
第一行输入一个整数 $ n $（$ 2 \le n \le 10^5 $）。  
接下来 $ n $ 行，每行输入一个字符串 $ s_i $（$ 1 \le |s_i| \le 10^6 $，所有字符串的总长度不超过 $ 10^6 $）。

**输出格式**  
输出一个整数，表示所有无序对的 LCP 长度之和。

**样例输入 1**  
```
3
abc
abd
abe
```

**样例输出 1**  
```
6
```

**样例输入 2**  
```
3
a
ab
abc
```

**样例输出 2**  
```
4
```
